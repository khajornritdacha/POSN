Sum Of Odd and Even
time limit per test
1 second
memory limit per test
64 megabytes
input
standard input
output
standard output

และวันนี้ก็มาถึง วันที่โจ้เจนเทสเคสเป็น ซึ่งนั่นหมายความว่าโจ้สามารถออกโจทย์เองได้แล้ว ว้าวๆๆๆๆๆๆๆๆๆ ซึ่งโจทย์ข้อนี้เป็นโจทย์ข้อแรกของโจ้ คุณซึ่่งต้องการตอบแทนน้ำใจที่โจ้จึงเลือกที่จะทำโจทย์ข้อนี้เพื่อตอบแทนเขา

งานของคุณ จงหาค่าผลรวมของเลขในตำแหน่งคู่ และตำแหน่งคี่ โดยข้อมูลนำเข้าตัวแรกนับเป็นตำแหน่งที่ $$$1$$$ และข้อมูลนำเข้าจะสิ้นสุดเมื่อรับเลข $$$0$$$ และไม่ต้องประมวลค่าเลข $$$0$$$

Input

มี $$$N$$$ ($$$1 \le N \le 10,000$$$) บรรทัด แต่ละบรรทัดประกอบด้วยจำนวนเต็ม 1 จำนวน คือ $$$x_i$$$ $$$(-100 \le x_i \le 100, \ x_i \neq 0,\ 1 \le i \le n)$$$ และรับประกันว่า $$$x_n = 0$$$ เสมอ

Output

มีบรรทัดเดียว ประกอบด้วยจำนวนเต็ม $$$2$$$ จำนวน คือ ผลรวมในตำแหน่งคู่ และผลรวมในตำแหน่งคี่ ตามลำดับ

Examples

Input
-70
-47
0
Output
-47 -70
Input
-48
-73
4
69
0
Output
-4 -44
