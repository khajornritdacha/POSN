# สรุปเรื่อง Array

## การประกาศ array

* สมมติจะประกาศ array ขนาด n ควรประกาศไว้ n+10 แทน(เผื่อไว้) และก็แนะนำให้ = {} ตลอดคิดซะว่าเป็น Good Practice
* เวลาประกาศถ้าจะให้ = {} ต้องใส่ขนาดตลอด 
* ถ้าเกิดจะให้มันเท่ากับอะไรซักอย่าง(เช่น = {1, 5, -3}) ก็เลือกที่จะใส่หรือไม่ใส่ขนาดก้ได้

```cpp
const int N = 100+10;
int arr1[N] = {}; // array 110 ช่องที่แต่ละช่องเป็น 0
int arr2[] = {5, 3, -1}; // array 3 ช่องโดยที่แต่ละช่องเป็น 5, 3, -1 ตามลำดับ
int arr3[10] = {5, 3, -1}; // array 10 ช่องโดยที่ 3 ช่องแรกเป็น 5, 3, -1 ตามลำดับ
```
    

## การรับค่า Array

* สมมติ array ชื่อ arr เราต้องรับค่าที่ละช่องเสมอ กล่าวคือรับ arr[i] ไปเรื่อย (ตามโค้ดด้านล่าง) 
* array เริ่มนับช่องแรกช่องที่ 0 แต่เราจะใช้ช่องไหนก็ได้ (โค้ดด้านล่างเริ่มใช้ช่องที่ 1)

```cpp
char arr[6] = {}; // เริ่มนับช่องที่ 0, 1, ... 5 (ทั้งหมด 6 ช่อง)
for (int i = 1; i <= 5; i++) {
    cin >> arr[i];
} // เริ่มรับค่าเก็บไว้ช่องที่ 1 ไปจนถึงช่องที่ 5

for (int i = 1; i <= 5; i++) {
    cout << arr[i] << " ";
}

/*
Example Input1:
a b 3 c f
Example Input2:
ab3cf
*/
```  


* จะมีเพียงกรณีเดียวที่เราสามารถรับค่าแบบไม่ใส่ช่องได้(ใส่ชื่อ array ไปตรงๆเลย) ก็คือกรณีที่เรารับ string(ตัวอักษรที่ติดกันโดยไม่มี space bar คั่น) เข้ามา 
```cpp
char arr[110] = {};
cin >> arr;
cout << arr;
/*
Example Input1:
arewr
Example Input2:
hello world
*/
```
จะเห็นว่าถ้าใส่ hello world เข้าไปจะปริ้นออกมาแค่ hello เพราะ cin มันตัดที่ space bar อ่ะ
